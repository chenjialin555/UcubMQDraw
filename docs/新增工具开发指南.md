# 新增工具开发指南

> **最常看的文档。** 新增一个生图工具时，按本清单操作。

**文档导航** → [README.md](./README.md)

## 一、要改哪些文件？

### 前端（4 处）

```text
□ frontend/src/tools/<tool-key>/
      XxxForm.vue          左栏表单
      XxxPreview.vue       中栏预览（可选）
      tool.ts              createForm / validate / buildRequest
□ frontend/src/tools/index.ts    tools 数组加一行
```

### 后端（2 处）

```text
□ backend/app/tools/<tool_key>.py    validate + build_task_dispatch_params
□ backend/app/tools/registry.py      tools 字典加一行
```

### 不要改

```text
✗ task_service.py 主流程
✗ task_api.py / upload_api.py
✗ taskStore.ts / ws.ts
✗ 数据库表结构（除非全项目级变更）
```

---

## 二、推荐开发顺序

```text
1. 列出表单字段
2. 写 XxxForm.vue
3. 写 tool.ts（createForm → validate → buildRequest）
4. Network 看 POST /tasks 的 params（对照 API接口.md）
5. 写后端 XxxTool.validate
6. 写 XxxTool.build_task_dispatch_params
7. USE_MOCK=true 看右栏进度
8. 再接真实 MQ
```

## 三、前端模板

目录结构：

```text
frontend/src/tools/my-tool/
├── MyToolForm.vue
├── MyToolPreview.vue    # 可选
└── tool.ts
```

`tool.ts` 骨架：

```ts
export const myTool = {
  key: 'my_tool',
  name: '我的工具',
  Form: MyToolForm,
  Preview: MyToolPreview,   // 无预览可 null

  createForm() { return { imageUrl: '', description: '' } },
  validate(form: any) { return form.imageUrl ? '' : '请上传图片' },
  buildRequest(form: any) {
    return {
      toolKey: 'my_tool',
      params: { prompt: form.description, imageUrl: form.imageUrl },
    }
  },
}
```

注册：`tools/index.ts` 的 `tools` 数组加一行。

**完整可复制代码** → [examples/前端工具模块示例.md](./examples/前端工具模块示例.md)

工具 ≤ 3 个时可用 `v-if` 替代动态组件 → [examples/HomePage完整示例.md](./examples/HomePage完整示例.md)

## 四、后端模板

`tools/my_tool.py` 骨架：

```python
class MyTool:
    key = "my_tool"
    name = "我的工具"

    def validate(self, params: dict) -> None:
        if not params.get("imageUrl"):
            raise ValueError("请上传图片")

    def build_task_dispatch_params(self, task_id, inner_task_id, params) -> dict:
        # task_id 仅存 DB；MQ 字段对齐 mq_contract.md
        return {
            "innerTaskId": inner_task_id,
            "modelName": params.get("modelName", "SDXL/Flux"),
            "prompt": params.get("prompt") or self.name,
            "width": params.get("width", 1024),
            "height": params.get("height", 1024),
            "batchSize": params.get("batchSize", 1),
            "refImageList": params.get("refImageList") or [],
            "timeout": params.get("timeout", 300),
        }
```

注册：`registry.py` 的 `tools` 字典加一行。

**完整可复制代码** → [examples/后端工具处理器示例.md](./examples/后端工具处理器示例.md)
